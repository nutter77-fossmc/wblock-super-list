# wBlock Super List

Aggressive ad/tracker/annoyance filters + userscripts for wBlock on macOS & iOS Safari.

## Installation

1. Open wBlock
2. Custom Filters → paste/import everything from /filters/
3. Userscripts → import both .user.js files
4. Enable everything and restart Safari

Warning: Extremely aggressive. Many sites will break.
